# 状态页功能使用指南

## 功能概述

状态页功能允许您创建公开的状态页面，无需登录即可查看监控项的状态信息，类似于Uptime Kuma的状态页功能。

## 主要功能

### 1. 多状态页支持
- 支持创建多个独立的状态页面
- 每个状态页可以展示不同的监控项组合
- 每个状态页有独立的URL访问地址

### 2. 公开访问
- 状态页完全公开，无需登录即可访问
- 支持自定义状态页标题和描述
- 实时显示监控项状态和统计信息

### 3. 自动刷新
- 状态页每30秒自动刷新数据
- 实时更新监控项状态和统计信息

### 4. 集成式管理
- 创建状态页时即可添加监控项
- 编辑状态页时可直接管理监控项
- 支持自定义监控项显示名称

## 使用方法

### 1. 访问状态页管理

1. 登录到酷监控管理界面
2. 在右上角点击状态页管理图标（图表线条图标）
3. 或直接访问 `/dashboard/status-pages`

### 2. 创建状态页

1. 在状态页管理界面点击"创建状态页"按钮
2. 填写以下信息：
   - **状态页名称**：例如"生产环境状态"
   - **URL标识符**：例如"production"（只能包含小写字母、数字和连字符）
   - **页面标题**：例如"生产环境服务状态"
   - **公开访问**：勾选此项使状态页公开访问
3. **添加监控项**（可选）：
   - 在创建时即可选择要添加的监控项
   - 可以为监控项设置自定义显示名称
   - 可以随时添加或移除监控项
4. 点击"创建状态页"按钮

### 3. 编辑状态页

1. 在状态页列表中点击"编辑"按钮
2. 可以修改：
   - 状态页基本信息（名称、URL标识符、标题、公开访问）
   - 添加新的监控项到状态页
   - 为监控项设置自定义显示名称
   - 移除不需要的监控项

### 4. 管理监控项

在编辑状态页时，您可以：

#### 添加监控项
1. 在"添加监控项"区域选择要添加的监控项（支持多选）
2. 可为每个监控项设置自定义显示名称（留空则使用原名称）
3. 点击"添加监控项"按钮批量添加

#### 移除监控项
1. 在"已添加的监控项"列表中找到要移除的监控项
2. 点击右侧的删除图标
3. 确认移除操作

#### 监控项显示
- 监控项按添加顺序显示
- 支持自定义显示名称
- 显示监控项类型和详细信息

### 5. 访问状态页

状态页的访问地址格式为：
```
http://your-domain/status/[slug]
```

例如：
- 状态页URL标识符为"production"时，访问地址为：`http://your-domain/status/production`
- 状态页URL标识符为"staging"时，访问地址为：`http://your-domain/status/staging`

### 6. 状态页界面

状态页界面包含以下信息：

#### 头部信息
- 状态页标题
- 页面描述

#### 统计信息
- 总计监控项数量
- 正常状态数量
- 故障状态数量
- 暂停状态数量
- 24小时可用性百分比

#### 监控项列表
- 监控项名称（支持自定义显示名称）
- 当前状态（正常/故障/暂停/未知）
- 最后检查时间
- 监控项描述（如果有）

#### 底部信息
- 最后更新时间
- 自动刷新提示

## 状态页管理

### 状态页列表
在状态页管理界面，您可以：
- 查看所有创建的状态页
- 查看每个状态页的监控项数量
- 查看状态页的访问权限（公开/私有）
- 查看状态页的创建时间

### 状态页操作
- **编辑**：修改状态页基本信息和监控项
- **删除**：删除状态页（需要确认）
- **查看状态页**：在新标签页中打开状态页
- **复制链接**：复制状态页访问地址到剪贴板

### 导航
- **返回监控面板**：点击左上角的"返回监控面板"按钮可返回主监控界面

## 技术特性

### 1. 数据安全
- 状态页只显示必要的状态信息
- 不暴露敏感配置信息
- 只显示公开的监控项

### 2. 性能优化
- 状态数据缓存，减少数据库查询
- 自动刷新机制，保持数据实时性
- 响应式设计，适配各种设备

### 3. 用户体验
- 简洁美观的界面设计
- 直观的状态指示器
- 友好的错误提示
- 加载状态提示
- 集成式管理界面，操作更便捷

## 使用场景

### 1. 生产环境状态页
- 展示生产环境所有服务的状态
- 供运维团队和业务方查看
- 快速了解系统整体健康状况

### 2. 客户状态页
- 为不同客户创建独立的状态页
- 只显示该客户相关的服务
- 提供客户专属的状态查看入口

### 3. 测试环境状态页
- 展示测试环境服务状态
- 供开发团队查看测试环境可用性
- 快速定位测试环境问题

## 注意事项

1. **URL标识符唯一性**：每个状态页的URL标识符必须唯一
2. **公开访问**：创建状态页时请确认是否需要公开访问
3. **监控项选择**：只添加需要公开显示的监控项到状态页
4. **数据更新**：状态页数据每30秒自动刷新，无需手动操作
5. **创建时添加监控项**：可以在创建状态页时直接添加监控项，也可以在创建后编辑添加

## 故障排除

### 状态页无法访问
1. 检查状态页是否设置为公开访问
2. 确认URL标识符是否正确
3. 检查服务器是否正常运行

### 监控项状态不更新
1. 确认监控项是否正常运行
2. 检查监控项是否已添加到状态页
3. 等待30秒自动刷新，或手动刷新页面

### 状态页显示错误
1. 检查浏览器控制台是否有错误信息
2. 确认网络连接正常
3. 尝试清除浏览器缓存后重新访问

### 无法添加监控项
1. 确认已有可用的监控项
2. 检查监控项是否已被添加到其他状态页
3. 刷新页面后重试
