# 设计系统与主题

本项目采用深浅双主题与紫色主色设计，关键规范如下。

## 主题
- 默认使用深色主题，支持深/浅切换

## 主色与辅助
- 主色: 深邃紫 `#6366F1`
- 渐变: `#6366F1 → #8B5CF6`

## 深色模式
- 页面: `#121827`
- 卡片: `#1E293B`（阴影: 0 4px 8px rgba(0,0,0,.25)，边框: 1px solid rgba(99,102,241,.15)）
- 次级背景: `#2D3748`
- 文字: 主 `#F1F5F9`，次 `#CBD5E1`，辅 `#94A3B8`

## 浅色模式
- 页面: `#F9FAFB`
- 卡片: `#FFFFFF`（阴影: 0 2px 10px rgba(0,0,0,.1)，边框: 1px solid rgba(99,102,241,.2)）
- 次级背景: `#F3F4F6`
- 文字: 主 `#111827`，次 `#374151`，辅 `#6B7280`

## 状态色
- 成功: `#10B981`
- 警告: `#F59E0B`
- 错误: `#EF4444`

## 组件要点
- 卡片: 阴影 + 紫色边框，悬停微动效
- 按钮: 主按钮紫色渐变；次按钮深/浅主题下分别为深灰/浅灰背景 + 紫色边框
- 输入与搜索: 深/浅主题背景 + 紫色边框，聚焦有紫色辉光
- 枚举按钮: 选中为渐变 + 白字 + 主色阴影；未选中为深色背景 + 次要文字 + 透明边框
- 图标: 紫色辉光；图标背景随主题小幅紫色透明背景
- 导航栏: 半透明 + 磨砂（backdrop-filter: blur）
- 滚动条: 与主题统一

## 响应式断点
- sm 640px, md 768px, lg 1024px, xl 1280px

## Tailwind 配置
- `tailwind.config.ts` 已定义 `darkMode: 'class'` 与颜色变量映射，详见源码
